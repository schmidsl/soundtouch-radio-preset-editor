# Changelog

## v0.4.0 – Initial public release

- Added Windows/Tkinter GUI.
- Added configurable `config.json`.
- Added preset display for slots 1–6.
- Added preset playback via `/select`.
- Added SoundCork status check.
- Added TuneIn station search through `/bmx/tunein/v1/search`.
- Added preview playback for search results.
- Added save-to-slot flow using `/storePreset`.
- Added automatic ZIP backup before changes.
- Added optional SoundCork `Presets.xml` sync.
- Added optional Docker Compose restart of the SoundCork service.
- Added Windows `.bat` and PowerShell start scripts.
