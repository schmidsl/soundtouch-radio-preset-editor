# Release v0.4.0

Initial public release of **SoundTouch Radio Preset Editor for SoundCork**.

This release is intended for users who already have a Bose SoundTouch speaker configured to use SoundCork and want a small local GUI for TuneIn radio presets.

## What works

- Read current Bose SoundTouch presets.
- Play existing presets.
- Search TuneIn stations through the local SoundCork BMX endpoint.
- Preview search results.
- Save selected TuneIn station to preset slot 1–6.
- Create automatic ZIP backup before changes.
- Sync SoundCork `Presets.xml` after save.
- Optionally restart SoundCork through Docker Compose.

## What this release does not do

- It does not install SoundCork.
- It does not switch a speaker to SoundCork.
- It does not restore backups.
- It does not edit Spotify/Deezer/Amazon/SiriusXM/DLNA/direct-stream presets.
